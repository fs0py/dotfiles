## stuff I use

- window manager: `i3-gaps`
- file manager: `ranger`
- terminal: `urxvt`
- compositor: `compton`
- text editor: `vim`
- bar: `polybar`
- application launcher: `rofi`

please, before using something that's here, read what's inside the files first to check everything will work properly.
